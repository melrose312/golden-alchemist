# Golden Alchemist

Golden Alchemist is a premium bilingual real estate website. It features a modern, responsive design and a luxurious gold-gradient aesthetic.

## Features

- **Bilingual Support**: Seamlessly switch between languages (powered by `i18next`).
- **Responsive Layout**: Designed with modern CSS Flexbox and Grid to work perfectly on desktop, tablet, and mobile devices.
- **Scroll Animations**: Engaging UI elements using `aos` (Animate On Scroll).
- **Modern Tech Stack**: Built with React and React Router.
